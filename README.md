# teemoShopping

#ENV

#Description
요약: 게임 구매 사이트를 위한 Rest API 제공 Backend

#Feature
- 로그인/ 회원가입
- JWTtoken 기반 Stateless 인증 구현
- RestAPI 를 통한 API 구현
- SpringSecurity의 Filter를 통해 보안 및 Jwt필터 구현
- WebClient을 이용한 카카오페이 API 연동
- 장바구니 구현 
- 쿠폰 구현
- 단건 결제 및 묶음 결제 구현
- 일부 환불 및 전체 환불 구현

#TechStack
1. Spring
  - SpringBoot
  - SpringSecurity
  - SpringValidation
  - SpringTest
2. H2
3. MYSQL
4. Util
  - Lombok
  - Jwt
  - Gson
